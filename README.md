# MRV-CPU

Mini RV32I CPU project with RTL, simulation, lint, and synthesis flows.

## Setup
```bash
# from repo root
source source_me.sh
# or:
export MRV_CPU_HOME=$(pwd)
```

## Repository Layout
- `docs/` : design/spec documentation and diagrams
- `lint/` : lint flows
- `rtl/` : CPU RTL sources
- `sim/` : simulation flows and tests
- `utils/` : helper scripts
